## Key facts

- SSH keys managed via 1Password SSH agent — no private keys on disk
- VM SSH config auto-committed here by VMDeployTools on every new VM deploy
- Pi-hole hardware (pihole1 + pihole2) documented here; API managed by pihole-flask-api
- K8s worker node inventory tracked here
- See [[architecture/homelab-overview|Homelab Overview]] for integration map
